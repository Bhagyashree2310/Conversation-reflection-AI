# Conversation Reflection AI

## Overview
This is a Streamlit-based AI application that analyzes user messages and helps improve communication by providing structured feedback.

## Features
- Tone detection
- Intent analysis
- Issue identification
- AI-generated improved message
- Sentiment analysis (Positive / Neutral / Negative)
- Dynamic tips based on message
- Feedback system (Yes/No + improvement input)
- History tracking

## Technologies Used
- Python
- Streamlit
- Groq API (LLaMA 3.1)
- Requests

## How to Run
1. Install dependencies:
pip install streamlit requests

2. Add your API key:
API_KEY = "your_api_key_here"

3. Run the app:
streamlit run app.py

## Note
API key is not included for security reasons.
